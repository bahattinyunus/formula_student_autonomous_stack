# Katkıda Bulunma Rehberi

Formula Student Autonomous Stack projesine katkıda bulunmak istediğiniz için teşekkür ederiz! Bu proje, Formula Student yarışmaları için açık kaynaklı bir otonom sürüş yığını geliştirmeyi hedefler.

## Nasıl Katkıda Bulunabilirsiniz?

1.  **Hata Bildirimi (Issues):**
    *   Karşılaştığınız hataları veya eksiklikleri GitHub Issues üzerinden bildirin.
    *   Hata bildirirken, hatayı yeniden oluşturmak için gerekli adımları ve ortam bilgilerinizi (ROS2 sürümü, OS vb.) ekleyin.

2.  **Özellik İstekleri:**
    *   Yeni bir özellik veya algoritma öneriniz varsa, bunu detaylı bir şekilde açıklayan bir Issue açın.

3.  **Pull Request (PR) Gönderme:**
    *   Projeyi fork'layın.
    *   Yeni bir özellik veya düzeltme için bir branch oluşturun (`feature/yeni-algoritma` veya `fix/hata-duzeltme`).
    *   Kodlarınızı yazın ve test edin.
    *   Commit mesajlarınızın açıklayıcı olmasına özen gösterin.
    *   Değişikliklerinizi fork'unuza push edin ve ana depoya bir Pull Request açın.
    *   PR açıklamasında yaptığınız değişiklikleri ve bunların nedenlerini belirtin.

## Kod Standartları

*   ROS2 standartlarına (REP) uygun kod yazmaya çalışın.
*   C++ kodları için Google C++ Style Guide, Python kodları için PEP8 takip edilmelidir.
*   Kodunuzu belgelendirmeyi (commenting/docs) unutmayın.

## İletişim

Sorularınız veya tartışmalar için [Issues](https://github.com/bahattinyunus/formula_student_autonomous_stack/issues) bölümünü kullanabilirsiniz.

Keyifli kodlamalar!
